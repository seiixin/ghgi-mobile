# GHGI Mobile Final - Modules 0 and 1 (Fullstack)

This package contains:
- `backend/` Node API (M0-M1)
- `mobile/` Expo mobile app (M0-M1)

## Order
1) Setup backend
2) Run backend on port 4000
3) Set `mobile/app.json` apiBaseUrl
4) Run mobile

## Database
Uses your existing Laravel DB (`users` table already exists). Backend creates:
- `devices`
- `refresh_tokens`
