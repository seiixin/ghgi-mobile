# GHGI Mobile (Expo SDK 54) - Modules 0-1

## Setup
1) Install:
   - `npm install`
2) Configure API:
   - `app.json -> expo.extra.apiBaseUrl`
   - Android emulator: `http://10.0.2.2:4000/api`
   - Real device: `http://<YOUR_PC_LAN_IP>:4000/api`
3) Run:
   - `npx expo start -c`
